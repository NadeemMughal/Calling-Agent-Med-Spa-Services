# Med Spa Services Voice Agent (Go High Level Integration)

## Overview
Voice agent for medical aesthetics clinic (e.g., Lumina Med Spa). Books dermal fillers, microneedling, skin boosters with complimentary consultations. Full GHL calendar and contact sync via **n8n**.

## Features
- Service details with pricing and duration
- Staff selection (e.g., Muhammad Nadeem)
- Complimentary consultation booking
- Precise spelling confirmation for names/emails
- Handles new and existing clients

## Tech Stack
- **Voice Agent**: Retell AI
- **CRM**: Go High Level
- **Workflows**: n8n

## Setup Instructions
1. Configure multiple GHL calendars (one per service) in n8n.
2. Import agent into Retell AI.
3. Set staff and location mappings.

## Usage
- Books treatments with accurate slot management and confirmations
